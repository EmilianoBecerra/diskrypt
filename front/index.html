Diskrypt

⦰ Envia archivos encriptados a cualquier parte del mundo ⦰

Arrastrá el archivo que quieras guardar.

  • El nombre del archivo en tu pc, se verá en el listado de archivos.
  • La contraseña que ingreses al subir el archivo, será la llave para desencriptarlo luego.
  • El archivo se descarga cuando se ingresa la contraseña correcta.
  • El archivo se borra automaticamente una vez se descargue.
